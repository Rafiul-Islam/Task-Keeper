A basic task keeper app using JavaScript DOM without database

## Features
 ⋅⋅* Add Task
 ⋅⋅* Edit Task
 ⋅⋅* Delete Task
 ⋅⋅* change task level with background color
 
## Technology
 ⋅⋅* HTML
 ⋅⋅* CSS
 ⋅⋅* Bootstarp
 ⋅⋅* JavaScript DOM
 ⋅⋅* Webpack
 ⋅⋅* Babel
 
 ## Install
 ---
 
 ### Clone
`git clone https://github.com/Rafiul-Islam/Task-Keeper/`
 
 ### NPM
 `npm install`
 
 run the project with this following command
 
 `npm start`
